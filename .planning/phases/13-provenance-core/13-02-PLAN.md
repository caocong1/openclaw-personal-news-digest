   - HTML anchor tags via `<a href="...">`
   - explicit labels `Original link:`, `original link:`, `Source:`, `source:`

   - extracted citations should be normalized to URLs where possible and stored as candidate `cited_sources`

   - include `title`, `normalized_url`, `content_snippet`, `tier_guess`, and extracted `cited_sources` in prompt input

   - when stripping HTML from RSS or extracted content, preserve anchor destinations as plain text `Original link: <url>` lines when a link target exists
   - do not discard upstream URLs from `content_snippet` during cleanup
   - keep this as a collection guarantee, not a later provenance workaround
